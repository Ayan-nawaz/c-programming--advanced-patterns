# C Programming – Advanced Patterns

A collection of advanced pattern printing programs in C. This repository includes star patterns, number patterns, and special pattern designs to strengthen logic building, nested loops, and programming fundamentals.

## Programs Included

1. Star Triangle Master (`star-triangle-master.c`)
2. Number Pyramids (`number-pyramids.c`)
3. Star Diamonds (`star-diamonds.c`)
4. Star Triangle Ultra Master (`star-triangle-ultra-master.c`)
5. Special Pattern (`special.c`)

## Concepts Covered

- Nested Loops
- Pattern Printing
- Star Patterns
- Number Patterns
- Pyramid Patterns
- Logic Building
- C Programming Fundamentals

## Requirements

- C Compiler (GCC, Turbo C, or any standard C compiler)
- Visual Studio Code or any C IDE

## How to Run

1. Compile the program:
   ```bash
   gcc filename.c -o output
   ```

2. Run the executable:
   ```bash
   ./output
   ```

## Learning Outcome

These programs help improve problem-solving skills, loop concepts, and pattern-printing techniques in C programming.
